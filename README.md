 Práctica de JavaScript – Funciones de Arrays



---

 Descripción

Este repositorio contiene la resolución de 16 ejercicios de JavaScript centrados en el uso de métodos de arrays como `filter`, `map`, `reduce`, `find`, `sort`, `forEach` e `includes`.

Cada ejercicio se encuentra en un archivo independiente (HTML o JS) en la raíz del repositorio.

---

 Ejercicios

Ejercicio 1: Números pares

**Archivo:** `ejercicio1.js`
Obtener los números pares de un array usando `filter`.
