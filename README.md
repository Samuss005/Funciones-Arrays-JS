# Funciones-Arrays-JS
Práctica de JavaScript – Arrays, Filter y Map

Alumno: Samuele Isidoro Moccia

 Descripción

Este repositorio contiene la resolución de 5 ejercicios de JavaScript centrados en el uso de arrays, métodos funcionales como filter y map, y manipulación de datos.

Cada ejercicio está implementado en un archivo JavaScript independiente o dentro de un archivo HTML según la entrega requerida, todos ubicados en la raíz del repositorio.

Ejercicios
Ejercicio 1: Números pares

Archivo: ejercicio1.js
Dado un array de números, usar filter para obtener únicamente los números pares.

 Ejercicio 2: Filtrar por propiedad de objeto

Archivo: ejercicio2.js
Filtrar un array de objetos productos para obtener solo aquellos con stock > 0.

 Ejercicio 3: Eliminar duplicados

Archivo: ejercicio3.js
Eliminar valores duplicados de un array usando filter e indexOf, manteniendo solo elementos únicos.

 Ejercicio 4: Precios con IVA

Archivo: ejercicio4.js
Usar map para calcular el precio final con un 21% de IVA. Redondear a 2 decimales utilizando toFixed(2) y convertir el resultado a número.

Ejercicio 5: Extraer propiedades con destructuring

Archivo: ejercicio5.js
Usar map junto con destructuring para crear un nuevo array de objetos simplificados con solo nombre y edad.

Notas importantes
Cada ejercicio debe estar en un archivo independiente.

Todos los archivos deben estar en la raíz del repositorio.

Se deben seguir las pistas proporcionadas para resolver cada ejercicio.

Se recomienda comprender la lógica antes de buscar ayuda externa.

Métodos de arrays (filter, map)
